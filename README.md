# 🎓 Online Kurs Platformu (Python OOP)

Bu proje, Python kullanılarak geliştirilmiş **komut satırı tabanlı (CLI)** bir kurs yönetim sistemidir.  
Eğitmen, öğrenci ve kurs yönetimi işlemleri OOP prensipleri ile tasarlanmıştır.


# 🚀 Özellikler

- Eğitmen ekleme ve listeleme
- Öğrenci ekleme ve listeleme
- Kurs oluşturma
- Öğrenciyi kursa kaydetme
- Öğrenciye ait kursları görüntüleme
- Eğitmen, öğrenci ve kurs silme
- Hatalı girişlere karşı güvenli input sistemi


# 🧠 Kullanılan Teknolojiler

- Python 3
- Object-Oriented Programming (OOP)
- CLI (Command Line Interface)
