# 🎓 ProKurs Admin Dashboard

Modern Python + CustomTkinter + SQLite tabanlı kurs yönetim sistemi.



# 🚀 Proje Açıklaması
ProKurs, eğitmen, öğrenci ve kurs yönetimini tek panel üzerinden yapmayı sağlayan masaüstü uygulamasıdır.



# ✨ Özellikler

# 🔐 Giriş Sistemi
- Admin login ekranı
- SQLite doğrulama sistemi
- Varsayılan giriş:
  - kullanıcı: admin
  - şifre: 1234
